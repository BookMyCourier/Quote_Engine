BookMyCourier Quote Engine - Version 1.3.0

Added
- Collection Date option with today's date selected by default.
- Collection Time option: Any time or Collected by set time.
- Delivery Option selector: Guaranteed same day or Within 2 days.
- Within 2 days delivery now reduces the calculated fee by 10%.
- Same-day delivery availability check. Same-day is disabled for today once the booking time is after 12pm.
- Collection and delivery options are passed through to the Book Now payment URL.

Changed
- Quote recalculates automatically when vehicle, address, collection date/time or delivery option changes.
- Price note now reflects whether the quote is same day or includes the 10% two-day discount.

Retained from 1.2.0
- Supplied van thumbnail images.
- Hidden customer-facing distance calculation.
- Automatic quote generation once addresses are entered.
- Book Now button sends customers to the configured payment URL with quote details.

BookMyCourier Quote Engine - Version 1.2.0

Added
- Replaced generated vehicle thumbnails with supplied BookMyCourier van images.

Changed
- Removed the top “Instant Courier Quote” lozenge.
- Vehicle cards now show vehicle image, vehicle name and use-case description only.
- Updated version number to 1.2.0 so WordPress can replace the existing plugin when uploading the ZIP.

Retained from 1.1.0
- Automatic quote generation once both addresses are entered.
- Distance remains hidden from the customer-facing quote panel.
- Book Now button sends customers to the configured payment URL with quote details in the query string.

Version 1.1.0
- Added vehicle illustration assets for Small Van, Medium Van, Large Van and Luton Van.
- Removed visible “From £...” pricing from vehicle cards.
- Removed visible distance display from the quote panel.
- Replaced manual quote button with automatic quote calculation after both addresses are entered.
- Added Book Now button that sends the customer to the configured payment URL with quote details in the query string.
- Added Book Now Payment URL setting.

Version 1.0.0
- Initial quote engine with Google Places and Routes API.

Version 1.4.0
- Renamed “Guaranteed same day” to “Same Day”.
- Added Next Day delivery option with 5% saving.
- Kept Within 2 days delivery option with 10% saving.
- Replaced specific collection time with ASAP collection or dated collection.
- Dated collection now uses AM/PM collection window instead of exact time.
